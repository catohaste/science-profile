About MRes

MRes in Modelling Biological Complexity

CoMPLEX, University College London

Starting in October 2016, I undertook an MRes (Master of Research degree) at CoMPLEX, UCL. CoMPLEX is a department specializing in bringing skills from the mathematical and computational sciences to problems in the life sciences.

The first term of the course is devoted to attending lectures aiming to improve both biological knowledge and quantitative skills. The second term is made up of three 6-week-long mini projects, whilst the summer is dedicated to a single 12-week project.

In addition to project work, I was required to complete 7 smaller tasks aimed at improving my transferable skills.

Summer Project

The aim of my summer project was to further develop an image analysis tool which aids in the study of both the localization and the gating function of the CFTR protein: the protein which, when mutated, causes cysitic fibrosis.

CFTR is a protein which acts as an plasma membrame ion channel. There are more than 200 mutations which are known to cause cystic fibrosis. Each of these mutations affects the function of the ion channel in different ways. My project has been aimed at finding a simple way of quantifying how the function of CFTR has been disrupted.

It is hoped that the image analsis tool I have helped to develop will be used to identify compounds improving the function of mutated CFTR, in discovering second site mutations which can resue CFTR's function and in further studying rare CFTR variants.

This project continued the work started in my third mini project.

Mini Projects

Project 1
Using computer simulations to study the effect of laminin and fibronectin on a biomechanical model of solid tumour growth

Project 2
Fitting a reinforcement learning model to data from a perceptual decision task in mice with asymmetric midbrain dopamine stimulation

Project 3
Development of an image analysis tool quantifying the efficacy of intra-cell trafficking of the CFTR protein

Transferable Skills

SQL Gene Ontology Task
Comparing chemosensation within the olfactory systems of four mammalian species using the SQL language to query the Gene Ontology database.

Summer Project Proposal

Poster Presentation

Referee Reports
Writing referee reports for two papers each developing methods to find correlations between phenotypes and genotypes.
